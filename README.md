## :newspaper:了解PAT
- 官方网址：[https://www.patest.cn/](https://www.patest.cn/)
- 浙江大学计算机程序设计能力考试（`Programming Ability Test`）
- 考试时间：每年三次，3，8，11月
- 甲乙两个等级，考试三个小时:sleeping:，成绩当场出，证书当场领:heart_eyes:。
- 甲级共四题，分数为：20、25、25、30。
- 乙级共五题，分数为：15、20、20、20、25。
